📘 Čo je display a prečo potrebujeme flex?

Display - Ako sa prvky zobrazujú

Každý HTML element má vlastnosť display, ktorá určuje, ako sa prvok zobrazí na stránke.

Najčastejšie hodnoty:

  • display: block; - Prvok zaberá celú šírku (napr. <div>, <p>, <h1>)
  • display: inline; - Prvok je len taký široký, ako jeho obsah (napr. <span>, <a>)
  • display: flex; - Flexbox režim - moderný spôsob vytvárania layoutov

Prečo Flexbox?

Pred flexboxom bolo ťažké vytvoriť jednoduché veci ako:

  • ✅ Tri tlačidlá vedľa seba s rovnakými medzerami
  • ✅ Obsah vertikálne vycentrovaný
  • ✅ Navigačné menu s rovnakým rozložením
  • ✅ Responzívne galérie, ktoré sa prispôsobia obrazovke

Flexbox to všetko rieši jednoducho! 🎉
